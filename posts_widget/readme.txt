Posts Widget Plugin for Hotaru CMS
--------------------------------
Created by: Nick Ramsay

Description
-----------
Displays lists of posts in widgets: Top, New, Upcoming, Top 7 Days, Top 30 Days, Top 365 Days and All-Time Best.

Instructions
------------
1. Upload the "posts_widget" folder to your plugins folder. 
2. Install it from Plugin Management in Admin.
3. Enable votes from Admin -> Vote
4. Enable the widget in Admin -> Widgets
