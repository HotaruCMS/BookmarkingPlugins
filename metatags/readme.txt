Metatags Plugin for Hotaru CMS
---------------------------------
Created by: shibuya246

Description
-----------
This plugin allows you to edit the metatags description. keywords for the main page to override the values in main_language file


Instructions
------------
1. Upload the "metatags" folder to your plugins folder. Install it from Plugin Management in Admin.
2. Go to Admin -> Plugin Settings -> metatags and edit the description, keywords, then save

Notes
-----
1.  The metatags will be shown for the top page only. Future versions may consolidate more meta tags on other pages as well
