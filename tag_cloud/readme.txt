Tag Manager Plugin for Hotaru CMS
--------------------------------
Created by: shibuya246

Description
-----------
With this plugin you manage the tags in your admin dashboard.

Instructions
------------
1. Upload the "tag_colors" folder to your plugins folder.
2. Install it from Plugin Management in Admin.
3. Edit the tag cloud settings in Admin -> Tag Manager
4. There are CSS and language files in the plugin folder that you can customize if you want to.
