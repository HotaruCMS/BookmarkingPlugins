Domain Results Plugin for Hotaru CMS
--------------------------------
Created by: Nick Ramsay

Description
-----------
A simple plugin to show a page of results for a given domain.

Instructions
------------
1. Upload the "domain_results" folder to your plugins folder.
2. Install it from Plugin Management in Admin.
3. Use it by entering one of these urls:

example.com/index.php?page=domain&domain=example2.com
example.com/domain/example2.com (friendly urls)
