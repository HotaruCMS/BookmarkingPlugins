Follow Plugin for Hotaru CMS
--------------------------------
Created by: shibuya246 (http://shibuya246.com)

Description
-----------
Basic Follower/Following plugin

Instructions
------------
1. Upload the "follow" folder to your plugins folder.
2. Install it from Plugin Management in Admin.
