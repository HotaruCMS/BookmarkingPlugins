Who Voted Plugin for Hotaru CMS
-----------------------------------
Created by: Nick Ramsay

Description
-----------
Display who voted on post pages.

Instructions
------------
1. Upload the "who_voted" folder to your plugins folder. 
2. Install it from Plugin Management in Admin. 
3. Edit how users are displayed in Admin -> Who Voted
