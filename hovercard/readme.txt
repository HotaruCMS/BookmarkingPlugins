HoverCard Plugin for Hotaru CMS
--------------------------------
Created by: Shibuya246

Description
-----------
Provides rollover hover cards for users.

Instructions
------------
1. Upload the "hovercard" folder to your plugins folder. 
2. Install it from Plugin Management in Admin.
3. Edit settings in Admin -> User Signin
